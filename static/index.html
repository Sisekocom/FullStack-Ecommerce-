Routes information

Routes HTTP methods Description
/ GET Render an index.html
/api/users/register POST Register a new user
/api/users/login POST User login
/api/users/profile GET Get user profile (protected)
/api/users/profile/update PUT Update user profile (protected)
/api/products GET Get list of all products
/api/products/:id GET Get details of a specific product
/api/products POST Add a new product (admin protected)
/api/products/:id PUT Update a product (admin protected)
/api/products/:id DELETE Delete a product (admin protected)
Note: Some routes are protected and require authentication. Admin routes require admin privileges.
